# HKMSP

**All-in-one open-source MSP dashboard:**  
A modern web platform for managed IT support—HKMSP streamlines client request intake, remote troubleshooting (desktop + video), chat, monitoring, and asset management, using proven open-source modules.

***

## Features

- Helpdesk Ticketing (client portal, real-time request intake)
- Remote Desktop/Support (live troubleshooting, screen share)
- Video Conferencing (client calls, team sync)
- Real-Time Chat (team and client conversations)
- Network and Device Monitoring
- Asset & Client Management
- Dashboard & Reporting
- Modular API-Driven Architecture

***

## Technologies & Open Source Modules

| Feature                  | Recommended Open Source Projects                                |
|--------------------------|----------------------------------------------------------------|
| Ticketing/Helpdesk       | osTicket, Zammad, Rocket.Chat (live chat + ticketing)          |
| Remote Desktop/Support   | MeshCentral, RustDesk, DWService                               |
| Video Conferencing       | Jitsi Meet, BigBlueButton                                      |
| Chat/Messaging           | Rocket.Chat, Mattermost, Zulip                                |
| Asset Management         | Snipe-IT, Netbox                                               |
| Network Monitoring       | LibreNMS, Nagios, Graylog                                      |
| Project Management       | OpenProject, Wekan                                              |
| Workflow Automation      | Node-RED, n8n (optional, for notifications/workflows)          |

***

## System Requirements

- Docker + Docker Compose (recommended for local dev and orchestration)
- Node.js (for web/UI modules, optional backend)
- Python3 (optional for some integrations and automation)
- Linux or MacOS server (Windows-compatible with some modules)

***

## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/headknod-art/HKMSP.git
cd HKMSP
```

### 2. Setup with Docker Compose

- Configure `.env` files and service configs according to included samples (coming soon).
- Run modular services together with a single command:

```bash
docker-compose up
```

### 3. Modular Integration

- Integrations for ticketing, video, and remote desktop detailed in `/integrations`
- API guides and endpoints documented in `/docs`
